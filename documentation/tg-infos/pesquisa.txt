A escala de cores é aplicada no arquivo netstf de acordo com o meteorologista, pois o arquivo contêm as frequências das temperaturas em Fanrenheit. Com essas informações em números aplicam-se as cores de acordo com escala utilizada pelo meteorologista/centro de pesquisa.

Passo-a-passo:
- Os dados são recebidos diretamente do satélite via antena que está conectada no servidor geonetcast, este servidor encontra-se na sede em SP. Nessa primeira etapa os arquivos netstf são processados como PNG e enviados para outros servidores.
Os arquivos PNG's são estáticos, ou seja, uma vez modificados não podem ser revertidos. Já os arquivos netstfs podem ser  modificados.
- O servidor Storage comunica com o servidor de Geoprocessamento, que é o servidor responsável por alimentar os produtos com as imagens.
